# Domain Atlas v0.1.0 Release Notes

`v0.1.0` is the first portfolio-ready release candidate for Domain Atlas. It
is prepared locally in this repository; no GitHub tag, remote Release, push,
or default-branch change is performed by this iteration.

## Included

- Guided discovery and source selection with provenance, authority, region, and
  official-entry explanations.
- Expert source control for URL, Markdown, and PDF ingestion.
- LLM Wiki workspace, concept dependencies, learning-path content, and
  Wiki-first cited QA.
- Deterministic public Demo, golden catalog evaluation, Docker delivery, and
  GitHub Actions regression workflow.

## Verification

Run the deterministic checks locally:

```bash
uv run python scripts/regression.py --fast
uv run python scripts/regression.py --e2e
uv run python scripts/regression.py --golden-demo-eval
uv run python scripts/regression.py --browser-e2e
```

Live provider checks remain opt-in and are not part of CI.
